You need to put your ffmpeg in /ffmepg,like /bin,/doc,/include,/lib,LICENSE.txt

"PyQt5" is needed:
    pip install PyQt5
    pip install PyQt5-tools

"PyQt-Fluent-Widgets" is needed:
    pip install "PyQt-Fluent-Widgets[full]"

If you want to use *.ui,launch your Qt Designer and open it:
    For how to use PyQt-Fluent-Widgets in Qt Designer,visit https://pyqt-fluent-widgets.readthedocs.io/en/latest

Launch with Main.py
